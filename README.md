# ShikimoriBot
Telegram-бот для поиска информации об аниме на сайте Shikimori и управления личным списком аниме через Telegram. Бот позволяет находить аниме по разным критериям, включая название, сезоны, жанры и т.д., просматривать подробности, а при авторизации – отмечать просмотренное, добавлять в планы и т.д.

👉 Запущенный бот:
https://t.me/FastShikimoriBot
### Конфигурация окружения
Перед запуском создайте .env на основе .env.template.

Обязательные переменные:
#### BOT_TOKEN
Токен Telegram-бота от @BotFather.
#### SHIKI_CLIENT_ID и SHIKI_CLIENT_SECRET
Данные OAuth2-приложения Shikimori.
Создаются здесь:
https://shikimori.one/oauth/applications
#### SHIKI_REDIRECT_URI
Адресс вашего публичного сервера
```
SHIKI_REDIRECT_URI="https://your-domain.com/shiki/callback"
```
Этот же адрес нужно указать в настройках OAuth-приложения на Shikimori в поле Redirect URL.

### Работа с ботом

* При первом запуске команды /start бот отправит приветственное сообщение и покажет основное меню (кнопки). В меню доступны кнопки для популярных функций:
* * Просмотр списка онгоингов текущего сезона, последних релизов, случайной сбалансированной подборки аниме и т.п. (обозначены emoji 📺, 🔥, 🎲 и др. в меню).
* * 🕓 История поиска – бот покажет список аниме, которые недавно заинтересовали пользователя.
* **Поиск аниме**: Бот настроен так, что любое сообщение пользователя, не являющееся командой, трактуется как поисковый запрос. Результаты поиска приходят списком с кнопками (inline-клавиатура); при нажатии на название бот покажет подробную информацию о выбранном аниме.
* **Авторизация через Shikimori**: По команде /login бот вышлет кнопку «🔑 Войти через Shikimori». При нажатии откроется браузер с OAuth2-разрешением на Shikimori[^1]. После подтверждения доступа, бот сохранит токены доступа в базе данных, и вы сможете пользоваться командами, связанными с вашим аккаунтом:
* * Добавлять аниме в список просмотренных, планируемых и т.д.
* * Ставить оценку аниме, отмечать прогресс просмотров.
* * Просматривать свой список (по кнопке 🌐 Shikimori в меню, бот загрузит ваш список «смотрю/просмотрено/запланировано» и выведет его).
* * **Управление списком**: После авторизации, при просмотре подробной информации о конкретном аниме, при нажатии «✅ Статус» будут работать дополнительные кнопки. Они позволяют изменить статус аниме в вашем списке Shikimori, отметить количество просмотренных серий или поставить оценку. Бот отправляет соответствующие запросы к API Shikimori и обновляет сообщение с информацией.
